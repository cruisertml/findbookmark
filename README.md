# 书签搜索工具 Chrome 扩展

一个功能强大的Chrome浏览器扩展，用于快速搜索、编辑和删除书签。

## 功能特性

1. **书签搜索**
   - 支持模糊匹配搜索
   - 可同时搜索书签标题和URL
   - 实时搜索，输入即显示结果

2. **书签编辑**
   - 修改书签名称
   - 支持键盘快捷键（Enter保存，ESC取消）

3. **书签删除**
   - 一键删除不需要的书签
   - 删除前确认提示

## 安装方法

1. 打开Chrome浏览器，进入扩展程序管理页面：
   - 方法1：在地址栏输入 `chrome://extensions/`
   - 方法2：点击右上角三点菜单 → 更多工具 → 扩展程序

2. 开启"开发者模式"（右上角开关）

3. 点击"加载已解压的扩展程序"

4. 选择本扩展的文件夹（包含manifest.json的文件夹）

5. 扩展安装完成后，点击浏览器工具栏的扩展图标即可使用

## 图标说明

扩展需要图标文件才能正常显示。请将以下尺寸的图标文件放置在 `icons` 目录下：
- `icon16.png` (16x16像素)
- `icon48.png` (48x48像素)
- `icon128.png` (128x128像素)

如果没有图标文件，可以：
1. 使用在线图标生成器创建
2. 使用任何图片编辑软件创建
3. 暂时使用占位图片（扩展仍可正常使用，只是没有图标显示）

## 使用方法

1. 点击浏览器工具栏的扩展图标
2. 在搜索框中输入关键词
3. 查看搜索结果
4. 点击"编辑"按钮修改书签名称
5. 点击"删除"按钮删除书签

## 技术栈

- Manifest V3
- Chrome Bookmarks API
- 原生JavaScript
- 现代CSS样式

## 注意事项

- 扩展需要"bookmarks"权限才能访问和修改书签
- 删除操作不可恢复，请谨慎操作
- 编辑书签名称时，空名称将被拒绝
